# Nexus Agent v0.6

A LangChain-powered AI agent with a CARE currency tool.

## Setup

1. Install dependencies:

pip install langchain langchain-groq langchain-core


2. Get a free API key from [Groq](https://console.groq.com)

3. Add your key to the script or set it as an environment variable:

set GROQ_API_KEY=your_key_here


4. Run:

python nexus_agent.py


## Features

- CARE currency calculator tool
- Conversational AI with tool-use capability
- Built with LangChain + Groq (Llama 3.3 70B)
